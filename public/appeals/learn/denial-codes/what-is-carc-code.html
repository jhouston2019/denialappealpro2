Denial Codes

What Is a CARC Code?

Every denied line on your remittance advice carries a two- or three-digit CARC code — the payer's standardized answer to why payment was reduced, shifted, or eliminated entirely.

Definition

A Claim Adjustment Reason Code — a standardized number on an EOB that tells a provider exactly why a claim or line item was adjusted or denied.

What a CARC Code Means

CARC stands for Claim Adjustment Reason Code. These codes are maintained under the X12 HIPAA standard and appear on every 835 remittance advice and patient EOB when a payer changes what they will pay on a claim. Each code maps to a specific reason — for example, code 97 means the service was bundled into another procedure, and code 50 means the payer rejected it for lack of medical necessity. CARC codes are always paired with a group code (CO, PR, OA, or PI) that tells you who is financially responsible for the adjusted amount.

Why CARC Codes Matter

Without CARC codes, every denial would arrive as a vague payment reduction with no actionable explanation. Revenue cycle teams rely on these codes to route claims to the right work queue — authorization denials go to precert staff, bundling denials go to coding, and timely filing denials go to claims submission. Industry data shows that just 5 CARC codes — including 97, 50, and 29 — drive more than 40% of all commercial claim denials. Identifying the code correctly is the first step toward recovery.

How CARC Codes Affect Your Claim

A CARC code determines whether you should appeal, rebill, collect from the patient, or write off the balance. A CO-97 adjustment means the payer considers the service included in another payment and you cannot bill the patient for that amount — your path forward is an appeal with modifier documentation or a corrected claim. A PR-1 code means the patient owes a deductible, which is not appealable but confirms the claim was processed. Misreading the group code attached to the CARC is one of the most common reasons providers either appeal unnecessarily or miss recoverable denials worth $200 to $2,000 per line item.

Always read the CARC code together with its group code (CO, PR, OA). The number tells you why payment changed; the group code tells you whether an appeal, a patient statement, or a write-off is the correct next step.

What to Do About a CARC Code

When you receive a remittance with a CARC code, look up the code definition, check for any accompanying RARC remark codes, and match the denial to your internal appeal playbook. Pull the original claim, the clinical documentation, and any authorization records relevant to that code. Submit a first-level appeal within the payer's deadline — typically 30 to 180 days from the denial date — with documentation that directly addresses the reason the CARC code describes. Frequently Asked Questions

Where do I find CARC codes on my EOB?

CARC codes appear in the claim adjustment or remark section of your remittance advice or EOB, usually next to each denied or reduced line item. Look for a two- or three-digit number paired with a group code such as CO, PR, or OA.

How many CARC codes are there?

The X12 standard defines more than 300 CARC codes. In daily billing, roughly 15 to 20 codes account for the majority of denials, including codes 97, 50, 29, and 197.

Is a CARC code the same as a denial reason?

A CARC code is the standardized denial or adjustment reason assigned by the payer. It tells you the category of the problem — bundling, medical necessity, timely filing — but you often need the accompanying RARC remark code for full context.

Can I appeal any CARC code denial?

Most CARC codes can be appealed when you have supporting documentation or can correct a billing error. Some PR-group codes reflect legitimate patient cost-sharing and are not appealable to the payer, though the underlying service denial may still be.
