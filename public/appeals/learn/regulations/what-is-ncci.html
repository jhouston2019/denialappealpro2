Regulations and Law

What Is NCCI?

The National Correct Coding Initiative is CMS's automated gatekeeper for procedure code pairs — flagging more than 800,000 edit combinations each quarter to block payment on codes that should not be billed together.

Definition

CMS's system of edits that prevent improper payment of procedure code combinations that should not be billed together.

What NCCI Means

NCCI stands for the National Correct Coding Initiative, a CMS program that maintains automated edits identifying procedure code pairs that should not be billed and paid separately on the same date of service for the same patient. The system includes procedure-to-procedure edits, medically unlikely edits, and add-on code rules. Each edit designates a column 1 code that receives payment and a column 2 code that is typically denied when billed together without a valid modifier override.

Why NCCI Matters

NCCI is one of the most common sources of Medicare claim denials and a major driver of revenue leakage for outpatient and physician practices. Coding teams that do not run NCCI checks before submission face predictable denials on the remittance advice, often with CARC code 97 indicating bundled service. Because CMS updates NCCI quarterly with thousands of new or revised edits, even experienced billers encounter new bundling conflicts after each effective date.

How NCCI Affects Your Claim

When a claim passes through Medicare's processing system, NCCI edits run automatically against every line item. If a column 2 code is billed with its paired column 1 code without an allowed modifier, the secondary line is denied or reduced to zero payment. Commercial payers frequently mirror these edits in their own claim scrubbing software, meaning the same code pair that fails NCCI on Medicare may also fail on a commercial remittance. A single missed edit can cost $150 to $3,000 per line depending on the procedure.

NCCI edits update quarterly on January 1, April 1, July 1, and October 1. Run your charge review against the current quarter's edit file before submission to avoid preventable bundling denials.

What to Do About an NCCI Denial

Identify the specific NCCI edit pair on your remittance and determine whether a modifier override is permitted for that edit. If the services were genuinely distinct — separate anatomic site, separate session, or distinct clinical purpose — append the appropriate modifier and resubmit or appeal with operative notes documenting the separation. If no modifier is allowed, the column 2 service may not be separately billable and should be written off unless clinical documentation supports a rare exception. Frequently Asked Questions

Does NCCI apply to commercial insurance claims?

NCCI edits are mandatory for Medicare and Medicaid claims. Many commercial payers adopt NCCI edits or similar bundling logic in their own claim editing systems, so NCCI rules frequently cause denials on non-Medicare claims as well.

What is an NCCI column 1 and column 2 edit?

Column 1 is the primary procedure code that receives payment. Column 2 is the secondary code that is bundled into column 1 and denied when billed together unless a valid modifier override applies.

How often does CMS update NCCI edits?

CMS publishes NCCI edit updates quarterly, typically effective January 1, April 1, July 1, and October 1 each year. Billing teams should review quarterly releases to avoid new bundling denials on previously billable code pairs.

Can NCCI bundling denials be appealed?

Yes. When services were genuinely distinct and performed at separate anatomic sites or on separate occasions, you can appeal with documentation supporting modifier use or clinical separation. Successful appeals require operative notes and clear evidence the bundled service was independent.
