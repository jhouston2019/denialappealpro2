Regulations and Law

What Is an NCCI Bundling Edit?

When two procedure codes land on the same claim, an NCCI bundling edit decides which one gets paid — and CARC code 97 on your remittance usually means the secondary code was swallowed by the primary.

Definition

A specific NCCI rule that prohibits separate billing of two procedure codes when one is considered a component of the other.

What an NCCI Bundling Edit Means

An NCCI bundling edit is a specific procedure-to-procedure rule within CMS's National Correct Coding Initiative that blocks separate payment for two CPT or HCPCS codes billed on the same date of service. The edit designates one code as the comprehensive column 1 procedure and the other as the component column 2 code. When the edit fires, the payer treats the column 2 service as already included in the payment for column 1 and denies or zeroes out the secondary line.

Why NCCI Bundling Edits Matter

Bundling edits are among the top three denial categories for outpatient physician claims, driving millions of dollars in write-offs annually across Medicare and commercial payers. They exist to prevent duplicate payment for work that is inherently part of a larger procedure — such as billing a separate incision code with a major surgical procedure. Revenue cycle teams that understand bundling logic recover payment that would otherwise be abandoned, while teams that treat every CARC 97 as non-appealable leave recoverable dollars on the table.

How an NCCI Bundling Edit Affects Your Claim

When a bundling edit applies, the remittance shows a CO-97 adjustment on the column 2 line, reducing payment to zero. You cannot bill the patient for the denied amount because the payer considers it included in the primary procedure payment. If the edit allows modifier override and the services were clinically distinct, appending modifier 59 or an appropriate X-modifier before submission can prevent the denial. Without a permitted modifier, the only path to payment is an appeal demonstrating the services were not components of each other — a high documentation bar that succeeds in roughly 20 percent of appealed bundling cases with strong operative notes.

A CO-97 bundling denial means you cannot balance-bill the patient for the denied amount. Your recovery path is a corrected claim with a valid modifier or an appeal with documentation proving the services were distinct.

What to Do About an NCCI Bundling Edit Denial

Look up the specific code pair in the current NCCI edit table and check whether modifier override is allowed. Review the operative report to confirm whether the services were performed at separate anatomic sites or during distinct sessions. If a modifier was omitted but clinically supported, submit a corrected claim. If a modifier was present and denied, file a first-level appeal with the operative note, highlighting the distinct clinical work performed. Frequently Asked Questions

What CARC code appears on NCCI bundling denials?

NCCI bundling denials most commonly appear with CARC code 97, indicating the benefit for this service is included in the payment for another service. The remittance may also reference an NCCI edit identifier in the remark section.

What modifiers bypass an NCCI bundling edit?

Common override modifiers include 59, XE, XP, XS, and XU when the edit allows modifier use. Each modifier has a specific clinical meaning, and using the wrong modifier can trigger audit risk even if payment is initially received.

Can both codes be paid on the same claim?

When an NCCI bundling edit applies and no valid modifier override is present, only the column 1 code receives payment. The column 2 code is denied as bundled. Both codes may be paid only when documentation supports a clinically distinct service and an allowed modifier is appended.

Should I rebill or appeal an NCCI bundling denial?

If you missed a valid modifier, correct the claim and resubmit. If the services were genuinely distinct and a modifier was already present, appeal with operative notes and documentation proving the procedures were independent. Rebilling without new documentation rarely succeeds.
