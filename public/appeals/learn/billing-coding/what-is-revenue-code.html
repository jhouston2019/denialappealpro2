Billing and Coding

What Is a Revenue Code?

On every hospital claim, a four-digit revenue code tells the payer which department or service category generated the charge — and mismatches trigger instant denials.

Definition

A four-digit code used on institutional claims (UB-04) identifying the type of service or accommodation billed.

What a Revenue Code Means

A revenue code is a four-digit identifier used exclusively on institutional claims submitted via the UB-04 form. Revenue codes classify charges by department or service category rather than by specific procedure. Code 0120 represents room and board for semi-private medical/surgical accommodation. Code 0450 represents emergency room services. Code 0300 represents laboratory services. Each revenue code line on a UB-04 claim is paired with a HCPCS or CPT procedure code, a service date, and a charge amount. Together, the revenue code and procedure code tell the payer what was done and where in the facility it occurred.

Why Revenue Codes Matter

Revenue codes drive how hospital and facility claims are categorized, priced, and audited. Payers use them to apply department-specific payment rules, validate that the procedure code matches the service location, and aggregate charges for DRG or APC grouping. The National Uniform Billing Committee maintains over 500 active revenue codes, and using the wrong code can shift a claim into an entirely different payment category. Hospital billing teams report that revenue code errors contribute to roughly 6% of institutional claim denials — a rate that translates to significant revenue leakage for high-volume facilities.

How Revenue Codes Affect Your Claim

On outpatient hospital claims, the revenue code determines which Ambulatory Payment Classification group applies to the service. An emergency room visit billed under revenue code 0450 with E/M code 99284 follows a different payment path than the same code under revenue code 0510 for clinic services. Inpatient claims use revenue codes to itemize ancillary charges that may or may not be included in the DRG payment. Payers reject claims when the revenue code is invalid, expired, or inconsistent with the billed procedure code. Revenue code 0001 is a generic catch-all that many payers reject outright, forcing rebilling with the correct department code.

Never use revenue code 0001 as a default. Assign the specific department code that matches where the service was rendered to avoid automatic payer rejection.

What to Do About Revenue Code Denials

When an institutional claim denies due to a revenue code issue, compare the billed code against the NUBC revenue code manual and your chargemaster mapping. Submit a corrected UB-04 with the accurate revenue code if the original was wrong. For denials where the revenue code was correct but the payer applied the wrong payment policy, file an appeal with documentation showing the service was rendered in the department indicated by the revenue code. Maintain a current revenue code crosswalk in your hospital billing system and audit new chargemaster entries against NUBC updates quarterly.

Frequently Asked Questions

Where do revenue codes appear on a hospital claim?

Revenue codes appear on the UB-04 institutional claim form in the revenue code column on each service line, paired with a HCPCS or CPT code and a charge amount.

What is revenue code 450?

Revenue code 450 identifies emergency room services on an institutional claim. It is one of the most frequently billed revenue codes in hospital outpatient departments.

Do revenue codes replace CPT codes on hospital claims?

No. Revenue codes classify the department or type of service, while CPT or HCPCS codes identify the specific procedure. Both appear on the same claim line for outpatient hospital services.

What happens if I use the wrong revenue code?

An incorrect revenue code can cause claim rejection, payment at the wrong rate, or denial when the code does not match the billed procedure or department.
