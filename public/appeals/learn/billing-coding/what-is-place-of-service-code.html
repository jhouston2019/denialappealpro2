Billing and Coding

What Is a Place of Service Code?

Two digits in Box 24B tell the payer exactly where care happened — and when they are wrong, the claim denies before anyone reviews the clinical work.

Definition

A two-digit code on a claim indicating where the service was rendered — errors in this code are a common cause of denials.

What a Place of Service Code Means

A place of service code is a two-digit identifier on professional claims that tells the payer where the medical service was physically performed. Code 11 means office, code 21 means inpatient hospital, code 22 means outpatient hospital, and code 02 means telehealth provided other than in the patient's home. The POS code appears in Box 24B of the CMS-1500 form on every service line. Payers use this code to apply site-specific payment rules, verify that the provider is credentialed for that location, and determine whether the service is covered under the patient's benefit plan for that setting.

Why Place of Service Codes Matter

Payment rates, coverage rules, and billing requirements vary significantly by site of service. A procedure performed in an office may pay at a different rate than the same procedure in a hospital outpatient department. Telehealth services require specific POS codes to qualify for coverage under payer telehealth policies. Industry data shows that place of service errors account for approximately 4% of all initial claim rejections — a preventable loss that adds rework cost and delays payment by an average of 14 days per affected claim.

How Place of Service Codes Affect Your Claim

Payers cross-reference the POS code against the provider's enrollment records, the billed CPT code, and the payer's site-of-service policies. Billing POS 11 for a service performed in a hospital outpatient setting may trigger a denial because the payer's system expects POS 22. Using POS 11 for telehealth when POS 02 or 10 is required results in coverage denials under many commercial plans. Some payers apply facility fees or site-neutral payment adjustments based on the POS code, meaning an incorrect code can reduce payment even when the claim is not fully denied.

Verify the POS code against the actual location of service for every claim line. Default POS settings in your practice management system are a frequent source of systematic denials.

What to Do About Place of Service Denials

When a claim denies due to a POS code mismatch, pull the encounter record to confirm where the service was actually rendered. Submit a corrected claim with the accurate POS code if the original was wrong. If the POS code was correct but the payer denied based on site-of-service policy, prepare an appeal with documentation showing the service was appropriately performed at that location. Audit your top POS codes quarterly and update system defaults when your practice adds new service locations or telehealth offerings. Frequently Asked Questions

What is place of service code 11?

Place of service code 11 indicates the service was performed in the provider's office. It is the most commonly used POS code for outpatient physician services.

What is place of service code 02?

Place of service code 02 indicates telehealth services provided other than in the patient's home. It replaced the former POS 11 modifier approach for many telehealth claims.

Can the wrong place of service code cause a denial?

Yes. Payers deny claims when the POS code does not match the billed procedure, provider type, or site of service on file. POS mismatches are among the top 10 preventable billing errors.

Where does the place of service code appear on a claim?

On a CMS-1500 professional claim, the place of service code appears in Box 24B next to each service line. On institutional claims, the equivalent is the type of bill and revenue code combination.
