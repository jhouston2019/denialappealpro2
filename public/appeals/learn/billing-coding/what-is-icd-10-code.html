Billing and Coding

What Is an ICD-10 Code?

The diagnosis code on every claim answers one critical question for payers: why did this patient need the procedure you are billing?

Definition

International Classification of Diseases code — the diagnosis code that justifies the medical necessity of the procedure billed.

What an ICD-10 Code Means

ICD-10 stands for the International Classification of Diseases, 10th Revision. These alphanumeric codes describe patient diagnoses, symptoms, and conditions with far greater specificity than the previous ICD-9 system. Code M17.11 identifies primary osteoarthritis of the right knee, while code J06.9 covers acute upper respiratory infection. Every professional and institutional claim requires at least one ICD-10 diagnosis code linked to each procedure or service line. Payers use these codes to determine whether the billed service is medically necessary and covered under the patient's benefit plan.

Why ICD-10 Codes Matter

Diagnosis coding is the primary gatekeeper for medical necessity determinations. A procedure without a supporting diagnosis code will deny automatically, and a vague or incorrect code can trigger the same outcome. The transition to ICD-10 expanded the code set from roughly 14,000 to over 70,000 diagnosis codes, giving payers more granular criteria for coverage decisions. Medical necessity denials tied to diagnosis coding represent approximately 17% of all commercial claim rejections, making ICD-10 accuracy essential for revenue integrity.

How ICD-10 Codes Affect Your Claim

Payers run automated edits that cross-reference your ICD-10 codes against the CPT or HCPCS codes on the claim. Local Coverage Determinations and National Coverage Determinations specify which diagnosis codes support payment for specific procedures. If your primary diagnosis does not appear on the payer's approved list, the claim denies regardless of clinical appropriateness. Specificity matters — billing M54.5 for low back pain when the documentation supports M54.41 for lumbago with sciatica, right side, creates both a compliance gap and a denial risk.

List the most specific ICD-10 code supported by your documentation as the primary diagnosis. Generic or unspecified codes are among the fastest paths to a medical necessity denial.

What to Do About ICD-10 Code Issues

When a claim denies for medical necessity, review the clinical documentation against the diagnosis code submitted. If a more specific code is supported, submit a corrected claim. If the original code was accurate, prepare an appeal with clinical notes, test results, and treatment history that demonstrate why the service was necessary. Maintain a diagnosis-to-procedure crosswalk for your most common services to prevent mismatches before submission. Frequently Asked Questions

How many characters are in an ICD-10 code?

ICD-10-CM diagnosis codes contain three to seven characters. The first three characters identify the category, and additional characters specify laterality, severity, and other clinical details.

What is the difference between ICD-10-CM and ICD-10-PCS?

ICD-10-CM codes describe diagnoses on all claim types. ICD-10-PCS codes describe inpatient hospital procedures and appear only on institutional claims.

Can I bill multiple ICD-10 codes on one claim?

Yes. Claims typically include a primary diagnosis and up to several secondary diagnoses. The primary code must support medical necessity for the billed procedure.

Why do payers deny claims for diagnosis coding errors?

Payers deny claims when the diagnosis code does not justify the procedure, is not specific enough, or is excluded from coverage under the patient's plan. CARC code 50 is the most common adjustment for medical necessity denials.
